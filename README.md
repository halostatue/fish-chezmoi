# halostatue/fish-chezmoi

A short description of this module for [fish shell][fish shell].

## Installation

Install with [Fisher][fisher]:

```fish
fisher install halostatue/fish-chezmoi@v1
```

### System Requirements

- [fish][fish] 3.0+

## Functions

### fish-chezmoi

```fish
$ fish-chezmoi example
example output
```
